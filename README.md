# perceptron999

Простой офлайн транскрибатор речи на русском языке с использованием библиотеки [Vosk](https://github.com/alphacep/vosk-api).

## Требования

* Python 3.8+
* Установленная библиотека Vosk
* Скачанная модель Vosk для русского языка

Установить зависимости можно командой:

```bash
pip install vosk
```

Модели Vosk можно загрузить с сайта <https://alphacephei.com/vosk/models>. Для русскоязычных записей подойдет модель `vosk-model-small-ru` или более крупные варианты.

## Использование

1. Сконвертируйте аудио в формат WAV (моно, PCM, 16 кГц).
2. Запустите скрипт:

```bash
python transcriber.py path/to/audio.wav path/to/model -o output.txt
```

Результат распознавания будет сохранен в файл `output.txt`. Если параметр `-o` не указан, текст будет выведен в консоль.

## Как это работает

`transcriber.py` загружает модель Vosk и последовательно считывает аудиофайл небольшими фрагментами. Для каждого фрагмента вызывается распознавание `AcceptWaveform`. По завершении обработки аудио формируется итоговая строка с полученным текстом.
